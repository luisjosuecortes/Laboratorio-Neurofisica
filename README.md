# Laboratorio de Neurofísica

Este es el repositorio oficial del **Laboratorio de Neurofísica**. Aquí se gestionan las aplicaciones y herramientas utilizadas para la investigación y desarrollo en el área de neurofísica.

---

## 🚀 Comenzando

Sigue estas instrucciones para configurar el proyecto localmente.

### Instalación

Primero, ejecuta el servidor de desarrollo:

```bash
npm run dev
# o
yarn dev
# o
pnpm dev
# o
bun dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.

Puedes comenzar a editar la página modificando `app/page.tsx`. La página se actualiza automáticamente a medida que editas el archivo.
